So you want to create your own documentation? Great!
Before you can do this you need to hop on over to 
sphinx https://www.sphinx-doc.org/en/master/usage/quickstart.html

Before you do anything:
> pip install sphinx or > uv add sphinx
> sphinx-quickstart or > uv run sphinx-quickstart 

After doing that, you should configure your source/conf.py file otherwise the 
src files wont show
